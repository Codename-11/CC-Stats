Fix for toast text overflowing window on long messages.

## What's Changed

### Bug Fixes
- **Toast text wrapping** -- long toast messages (like update notifications) now wrap properly instead of extending past the window edge. Layout changed from horizontal StackPanel to Grid with constrained MaxWidth.

## Install

Download `CCStats-v0.4.1-win-x64.exe` from the assets below and run it. Self-contained -- no .NET runtime needed.

**Requirements:** Windows 10 (build 17763) or later.
